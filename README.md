Resolución de los ejercicios de la Actividad 2 de Taller de Lenguaje.
Requisitos

Python 3.10 o superior

Instalación

Clonar el repositorio:

git clone https://github.com/justinacer-hub/Practica2.git
cd Practica2

Crear y activar el entorno virtual:

python -m venv .venv
.venv\Scripts\activate        # Windows
source .venv/bin/activate     # Linux / Mac

Instalar dependencias:

pip install -r requirements.txt
Ejecución
Iniciar Jupyter desde la raíz del proyecto:
jupyter notebook
Luego abrir notebooks/10CompetenciaCocina.ipynb y ejecutar las celdas en orden.
